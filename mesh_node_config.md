# 📟 Meshtastic Node Setup

To receive data from the S3 Tracker, configure your in-car Meshtastic Radio:

1. **Serial Module:** - Enabled: `ON`
   - Baud Rate: `38400`
   - Mode: `SIMPLE`
2. **Position Settings:**
   - Fixed Position: `OFF` (Allow the S3 to provide the coordinates).
3. **Wiring:** Ensure a **Common Ground** between the S3 and the Mesh Node.
