# 🔌 PNWC Pro: Physical Wiring Diagram (T-SIM7670G-S3)

### 1. SW-420 Vibration Sensor
- **VCC:** 3.3V Pin
- **GND:** GND Pin
- **DO (Digital Out):** **GPIO 1**

### 2. LD2410 Radar Sensor
- **VCC:** 5V Pin (VBUS)
- **GND:** GND Pin
- **TX:** **GPIO 3**
- **RX:** **GPIO 2**

### 3. Antennas
- **GPS Puck:** IPEX Connect (Top Port) -> Windshield.
- **LTE Stick:** IPEX Connect (Main Port) -> Dash Interior.

### 4. Power
- **Battery:** Integrated 18650 Holder.
- **Maintenance:** USB-C Port.
